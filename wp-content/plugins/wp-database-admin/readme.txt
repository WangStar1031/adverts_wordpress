== Description ==

"Wordpress Database Administrator" allow you to Access your wordpress database in the Admin panel. Simple Plugin inspired  from PhpMyadmin.

Major features of "Wordpress Database Administrator":

* It Allow you to access wordpress Database form Admin panel 
* Admin can easily Retrieve, Insert, Update, Delete Data from table With Query Builder. 
* Admin can Also Execute Custom Query in database.
* Easily create query with Query Builder

== Installation ==

1. Upload "Wordpress Database Administrator" to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3.  Click on the  "Tool > DB Admin" and  Access Wordpress Database

1, 2, 3: You're done :)
